# 🏀 NBA Card Game

Projeto desenvolvido em Java inspirado no clássico Super Trunfo, onde o jogador enfrenta a máquina utilizando cartas de grandes estrelas da NBA.

## 📖 Sobre o Projeto

O objetivo deste projeto foi praticar conceitos fundamentais de Programação Orientada a Objetos (POO), estruturação de projetos Java e lógica de programação.

O jogador recebe uma carta aleatória e escolhe um atributo para disputar contra uma carta sorteada para a máquina. O vencedor é definido com base nos valores dos atributos escolhidos.

---

## 🚀 Funcionalidades

- Listagem de jogadores da NBA
- Cartas geradas aleatoriamente
- Escolha de atributos para batalha
- Comparação de atributos
- Determinação automática do vencedor
- Interface via terminal
- Organização em pacotes
